# siseai.online
Automated Job Tracker powered by Artificial Intelligence


For college students seeking internships or full-time job offers, juggling demanding classes and deliberating job searching can be overwhelming and challenging.
Areas such as;

i. keeping track of job updates, 
ii. optimizing resumes for specific roles, and 
iii. getting automatic job recommendations

are painstaking for these students. Sise.ai is a system that helps students stay on top of these issues in a few steps. 

Connect. Authenticate. Track. Optimize. Apply.
